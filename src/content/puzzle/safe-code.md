Safe codes in Resident Evil Requiem are typically hidden in nearby documents, notes, or environmental clues. Always check the room containing the safe for a memo with the combination.

## Common Safe Locations

### Care Center Reception Safe

- **Location**: Main reception desk, East Wing 1F
- **Clue**: The code is written on a sticky note behind the desk.
- **Typical Reward**: Handgun ammo and a healing item

### Director's Office Safe

- **Location**: Director's Office, West Wing 2F
- **Clue**: Check the calendar on the wall for a circled date.
- **Typical Reward**: Gunpowder and a collectible charm

### Storage Room Safe

- **Location**: Storage Room, East Wing B1
- **Clue**: The combination is split across three memos in the room.
- **Typical Reward**: Shotgun shells and a herb

## Rewards

Safes usually contain handgun ammo, healing items, collectible charms, or crafting materials. They are worth opening for the extra resources, especially on higher difficulties.

## Notes

If you are playing a randomized run, write down the code when you find it. There is no universal code that works for every playthrough. Some safes are optional and can be skipped if you are low on inventory space.
